Insecure Frontpage extensions configuration #69

Description

Incorrect permissions and/or missing authentication were discovered on FrontPage extensions used for publishing content.

A cyber-criminal might use this vulnerability to deface web sites and potentially gain remote code execution on the server.

Fix guidance (10 minutes)

  1. Change the permissions on directories and files accessible via IIS
  2. Setup authentication and authorization for FrontPage extension access
